## 🧑‍💼 OutOfWork
Unity 숙련 주차 팀프로젝트 4라가기 힘들조의 팀프로젝트 입니다.

## 💁‍♂️ 소개
### 제한 시간안에 퇴근을 막는 로봇들을 물리치고 1층에 도달하여 퇴근에 성공하자

## ⏰ 개발 기간
- 24.02.08 ~ 24.02.16

## 👨‍👨‍👧‍👦 멤버
- 송상화(팀장) : GameManager
- 김유원 : SkllTree, Sound
- 김상민 : UI
- 신채윤 : Unit , MainStage

## 💻 개발 환경
- Engine : Unity 2022.3.2f1
- Language : C#
